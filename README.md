# Sistema de Estoque

Sistema simples de controle de estoque desenvolvido em Python.

## Funcionalidades

- Adicionar produto
- Listar produtos
- Buscar produto
- Remover produto

## Como executar

```bash
python estoque.py
```

## Tecnologias

- Python 3
